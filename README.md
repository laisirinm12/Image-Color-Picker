# Image Color Picker

This is a simple **Image Color Picker Web Application** built using **HTML, CSS and JavaScript** with the **Canvas API**.  
It allows users to upload an image and interactively pick colors from any pixel in the image.

The application supports **hover color preview, click-to-lock color selection, HEX/RGB display, and copying color values** while maintaining a clean **developer-tool style interface** with a workspace background.

## Features

- Upload an image from the local system
- Detect pixel colors directly from the image
- Live color preview while hovering over the image
- Click on a pixel to **lock the selected color**
- Display selected color in **HEX and RGB format**
- Color preview box
- Floating tooltip showing color near the cursor
- Copy HEX value to clipboard
- Responsive canvas that maintains image **aspect ratio**
- Clean developer-tool inspired UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Canvas API

## Purpose

This project was created as a small **frontend development exercise** to practice:

- Working with the **HTML Canvas API**
- Extracting **pixel data from images**
- Converting **RGB values to HEX format**
- Handling **mouse events and user interactions**
- Designing a clean and interactive developer utility interface
